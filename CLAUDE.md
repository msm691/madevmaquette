# Contexte du Projet : MADEV Pass
Application web : Carte étudiante numérique de Vienne et réseau de commerçants partenaires.
- Interface 1 : Étudiant (QR code, annuaire, favoris).
- Interface 2 : Commerçant (Gestion des offres, scan QR).
- Interface 3 : Admin (Modération, statistiques).
